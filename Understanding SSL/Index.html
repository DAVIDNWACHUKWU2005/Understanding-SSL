Understanding SSL & TLS

Basics about website security

Questions and Answers

What is SSL?

SSL stands for Secure Sockets Layer. It's a technology that encrypts the connection between your browser and a website, ensuring that any data you share stays private and secure.

What is TLS?

TLS, or Transport Layer Security, is the updated version of SSL. It's a more secure and efficient way to protect data during online communication.

How can I tell that my webpage is secure?

Look for a lock icon next to the website's address and make sure the URL starts with "https://". The "s" means the site is secure and using encryption.

What does it mean to be secure?

A secure website ensures that the data you send is protected from hackers. This is done through encryption, which scrambles your data so only the intended recipient can read it.

What is encryption and why do I care?

Encryption is a way to protect your data by converting it into a code that only authorized parties can decipher. It’s important because it keeps your information safe from hackers when you’re online.

Do the prefixes HTTP or HTTPS matter when we're talking about secure websites?

HTTP is not secure, while HTTPS (with the "s" at the end) means the website is using encryption to protect your information.

What is an SSL certificate authority?

An SSL certificate authority (CA) is a trusted organization that verifies the identity of websites. They issue SSL/TLS certificates that enable encrypted communication and verify that a website is legitimate.
